## Paper #1: Helical parafermionic hinge modes in three-dimensional second-order topological superconductors

**Authors**: Sarthak Girdhar, Viktoriia Pinchenkova, Even Thingstad  
**arXiv**: [2604.07313](https://arxiv.org/abs/2604.07313)  
**Key words**: topology, superconductivity, majorana, parafermion, hinge modes  
**Year**: 2026

### Summary

A model of 3D helical second-order topological superconductor formed by weakly coupled Rashba nanowires. The system hosts gapless helical Z₂q parafermionic modes along 1D hinges, with Majorana hinge modes in the noninteracting limit (q=1).

### Core Concepts

- **Second-Order Topological Superconductor**: Gapped bulk and surface, but gapless hinge states
- **Rashba Nanowires**: Spin-orbit coupled nanowires with helical properties
- **Parafermions**: Generalization of Majorana fermions with Z₂q exchange statistics
- **Helical Hinge Modes**: Counter-propagating modes protected by symmetry

### Key Innovations

1. **3D Helical SOTSC**: First construction using coupled Rashba nanowire array
2. **Parafermionic Hinge Modes**: Gapless Z₂q parafermions along closed hinge paths
3. **Tunable Trajectory**: Hinge mode path controlled by interwire coupling hierarchy
4. **Majorana Limit**: Recovers Majorana hinge modes when q=1 (noninteracting)

### Technical Approach

- Array of weakly coupled Rashba nanowires with superconducting pairing
- Bulk and surface gap analysis for parameter regime identification
- Hinge state characterization using effective edge theory
- Symmetry protection analysis for parafermionic modes
