## Paper #1: $\mathbb{Z}_{2q}$ parafermionic hinge states in a three-dimensional array of coupled nanowires

**Authors**: Sarthak Girdhar, Viktoriia Pinchenkova, Even Thingstad  
**arXiv**: [2604.07313](https://arxiv.org/abs/2604.07313)  
**Key words**: topology, superconductivity, majorana, parafermion, hinge modes  
**Year**: 2026

### Summary

A model of 3D helical second-order topological superconductor formed by weakly coupled Rashba nanowires. The system hosts gapless helical $\mathbb{Z}_{2q}$ parafermionic modes along 1D hinges, with Majorana hinge modes in the noninteracting limit ($q=1$).

### Core Concepts

- **Second-Order Topological Superconductor**: Gapped bulk and surface, but gapless hinge states
- **Rashba Nanowires**: Spin-orbit coupled nanowires with helical properties
- **Parafermions**: Generalization of Majorana fermions with $\mathbb{Z}_{2q}$ exchange statistics
- **Helical Hinge Modes**: Counter-propagating modes protected by symmetry

### Key Innovations

1. **3D Helical SOTSC**: First construction using coupled Rashba nanowire array
2. **Parafermionic Hinge Modes**: Gapless $\mathbb{Z}_{2q}$ parafermions along closed hinge paths
3. **Tunable Trajectory**: Hinge mode path controlled by interwire coupling hierarchy
4. **Majorana Limit**: Recovers Majorana hinge modes when $q=1$ (noninteracting)

### Technical Approach

- Array of weakly coupled Rashba nanowires with superconducting pairing
- Bulk and surface gap analysis for parameter regime identification
- Hinge state characterization using effective edge theory
- Symmetry protection analysis for parafermionic modes

## Paper #2: Programmable higher-order nonequilibrium topological phases on a superconducting quantum processor

**Authors**: Haoran Qian, Ming Gong, Jiahui Zhang, Shaojun Guo, et al. (Pan Jianwei's group)  
**Journal**: Science (2025)  
**DOI**: [10.1126/science.adp6802](https://doi.org/10.1126/science.adp6802)  
**Key words**: topology, higher-order-topological-phase, Floquet, superconducting-qubit, quantum-simulation, chiral-density  
**Year**: 2025

### Summary

This study demonstrates both equilibrium and nonequilibrium higher-order topological phases (HOTPs) using a 66-qubit Zuchongzhi 2.0 superconducting quantum processor. The authors implemented >50 cycles of Floquet operators on a 6×6 qubit array and introduced a universal detection method based on chiral density dynamics to identify Floquet corner topological invariants and π-energy topological corner modes.

### Core Concepts

- **Higher-Order Topological Phases (HOTPs)**: Topological phases where boundary states emerge at (d-2) or lower dimensions, e.g., corner modes in 2D systems
- **Floquet Topological Phases**: Nonequilibrium topological phases in periodically driven systems, characterized by quasienergy spectrum with 2π periodicity
- **Second-Order Topological Phases (SOTPs)**: Gapped bulk and edges, but gapless corner states
- **Chiral Density**: A measurable quantity used to detect topological invariants via dynamics of the time-evolution operator
- **π-Energy Corner Modes**: Topological corner modes appearing at quasienergy π in Floquet systems
- **Floquet Topological Mirror Winding Numbers (Z₀, Z_π)**: Invariants characterizing nonequilibrium SOTPs based on the Floquet operator

### Key Innovations

1. **First Experimental Realization of 2D Nonequilibrium HOTPs**: Using a programmable superconducting quantum processor with Floquet circuits
2. **Universal Chiral Density Detection Method**: A novel approach to extract Floquet corner topological invariants by measuring chiral density dynamics
3. **>50 Floquet Operator Cycles**: Demonstrating long-time coherent dynamics of topological states on a 36-qubit array
4. **Direct Measurement of Topological Invariants**: Extracting tr(Uⁿ) from chiral density to characterize Floquet topological phases

### Technical Approach

- **Device**: 66-qubit Zuchongzhi 2.0 superconducting quantum processor (6×6 subset used)
- **System**: Bose-Hubbard model with tunable qubit frequencies and couplings
- **Floquet Circuit Design**: Two-layer quantum circuits implementing intracell and intercell couplings
- **Detection Protocol**: 
  - Measure chiral density dynamics ⟨Γ(t/2)⟩
  - Extract trace of Floquet operator tr(Uⁿ) via Fourier transformation
  - Compute local density of states (LDOS) and Floquet corner topological invariants
- **Phase Characterization**: Mapping out topological phase diagram using Floquet winding numbers (Z₀, Z_π)
