{\LARGE\bfseries\color{ustcblue} 使用说明}
\end{center}

\vspace{6pt}

\begin{insightbox}[本文档的定位]
不同于《中科大物理博士考试备考资料》的百科全书式汇总，本文档聚焦于\key{物理模型}的\key{考前练手}。

核心目标：
\begin{itemize}
    \item 快速识别考题对应的物理模型
    \item 掌握每个模型的核心公式与边界条件
    \item 明确每个模型的常见训练点与考法变体
    \item 建立模型之间的关联网络，实现知识迁移
\end{itemize}
\end{insightbox}

\vspace{4pt}

\begin{drillbox}[阅读指南]
每个模型按统一格式组织：
\begin{enumerate}[label=\textbf{\arabic*.}]
    \item \textcolor{modelteal}{\textbf{模型框}}：模型的精确定义、哈密顿量、边界条件
    \item \textcolor{successgreen}{\textbf{公式框}}：必须背诵的核心公式（零含糊）
    \item \textcolor{drillblue}{\textbf{训练点框}}：该模型在考试中可能出现的考法、变体、组合
    \item \textcolor{alertred}{\textbf{技巧框}}：快速识别标志、解题捷径
    \item \textcolor{errorred}{\textbf{易错框}}：常见失误、边界条件遗漏、符号陷阱
    \item \textcolor{royalpurple}{\textbf{洞察框}}：物理图像、与相似模型的对比、推广方向
\end{enumerate}
\end{drillbox}

\vspace{4pt}

\begin{cheatbox}[颜色速查]
\begin{tabular}{ll}
\textcolor{modelteal}{$\blacksquare$} & 模型定义（必须精准记忆） \\

\end{tabular}
\end{cheatbox}
